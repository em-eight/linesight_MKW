
## MKW Scripting Modules

This directory contains a plethora of python scripts created to interface programatically with Mario Kart Wii. It is based on the Dolphin PyCore fork which itself is based on Felk's scripting fork. These modules have been modified to work specifically with Felk's fork, bypassing certain additions of PyCore to facilitate screen grabbing capabilities.

The git repository for some of these modules can be found [here](https://github.com/TASLabz/mkw-scripts).
The full plethora of scripts added by the PyCore team can be found in the [PyCore git repository](https://github.com/Blounard/dolphin-pycore).

Usage examples can be found in the RMC folder in `_draw_info_display.py` and `_draw_input_display.py` along with additional TASing tools and ghost management programs.
